About Earthmover:

Earthmover is building products that solves the biggest storage and compute challenges of our time for users in the deep sciences - think climate, earth observations, biotech, machine learning, and AI.

Our first product, Arraylake, is a data lake platform for multi-dimensional scientific array data. It enables users to manage and process array data in open standard cloud-native formats, with features like versioning, cataloging, collaboration and access control on top. Arraylake was built to integrate seamlessly with the open source scientific Python ecosystem, and provides an interface via the Zarr project. Through the Zarr interface, scientific data users interact with data via libraries such as Xarray, AnnData, Cooler, Napari, and others.

Our team has deep experience developing and maintaining open source software libraries across the scientific Python ecosystem. Our open source stack:

  • Xarray is the focus of this hire. It provides a high-level data model and API for working with labeled N-dimensional arrays. Think of it it like Pandas for multidimensional data.
  • Zarr is a community developed data format for chunked, compressed multidimensional arrays. Zarr-Python is the interface to Zarr used by Xarray, Arraylake, and many other libraries.
  • Both Xarray and Zarr are built to integrate with the rest of the Scientific Python ecosystem. Think NumPy, SciPy, Scikit-learn, Dask, Matplotlib, etc.
